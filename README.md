# GRA Website

This is a static, self-contained German Rifle Association (GRA) website package.

Open `index.html` in a browser to start.

## Included Pages

- `index.html` - main GRA site with popup, newsletter flow, dropdown navigation, More drawer, chat UI, German content sections, finder table, events, and footer sources.
- `member-services.html` - member services/login page.
- `find.html` - "Find your GRA" style page with accordions and local results.
- `join.html` - membership/special-offers/join flow with forms and payment UI.
- `search.html` - dynamic local search results page.

## Technical Notes

- The site is static HTML, CSS, and JavaScript.
- Newsletter, login, chat, finder, order, and payment functions stay in the browser.
- No email, payment, login, SMS, or form data is transmitted.
- Legal content is editorial context only and not legal advice.
