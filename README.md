# GRA Website Demo

This is a static, self-contained demo inspired by the provided NRA screenshots, transformed into a fictional German Rifle Association (GRA) website.

Open `index.html` in a browser to start.

## Included Pages

- `index.html` - main GRA site with popup, newsletter flow, dropdown navigation, More drawer, chat simulation, German content sections, finder table, events, and footer sources.
- `member-services.html` - member services/login simulation.
- `find.html` - "Find your GRA" style page with accordions and local demo results.
- `join.html` - membership/special-offers/join flow with simulated forms and payment UI.
- `search.html` - dynamic local search results page.

## Important Demo Notes

- This site is not affiliated with NRA, any real GRA, any German authority, or any manufacturer.
- Newsletter, login, chat, finder, order, and payment functions are frontend simulations only.
- No email, payment, login, SMS, or form data is transmitted.
- Legal content is editorial context only and not legal advice.
